Voice Scam Shield - Backend Setup
---------------------------------

1. Create virtual environment:
   python -m venv venv
   source venv/bin/activate  (Linux/Mac)
   venv\Scripts\activate     (Windows)

2. Install dependencies:
   pip install -r requirements.txt

3. Set your OpenAI API key (only if you want online mode):
   Windows:
       setx OPENAI_API_KEY "your_api_key_here"
   Linux/Mac:
       export OPENAI_API_KEY="your_api_key_here"

4. Run backend:
   uvicorn main:app --reload

5. Access backend at:
   http://127.0.0.1:8000

6. API Endpoints:
   POST /upload-audio   -> Upload audio file (.wav/.mp3) for detection
   GET  /reports/{id}   -> Get scam detection report
   GET  /audio/{file}   -> Download audio file
   WS   /ws             -> WebSocket updates
